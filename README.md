 # My Student Portfolio 
Asimple project to learn Git.
Goals: Become Devops Expert! 
# Summary Of the Tasks
This task taught me how to use GitHub and Git for version control. I began by making my first commit and setting up a local repository. I then uploaded my code online and linked my project to GitHub. I worked on making branches, adding new files, and combining them with the main branch.  I also worked with pull requests and acquired conflict resolution skills. I also investigated key Git operations including using stash, viewing commit history, and undoing changes. Ultimately, I finished the cycle by creating an issue and connecting it to a pull request.
